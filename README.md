# KMobin555 — Portfolio

This is a simple static portfolio site scaffold. It contains:

- `index.html` — the main page with hero, about, projects, and contact sections.
- `styles.css` — styles and responsive layout.
- `script.js` — small client-side interactions (contact form demo).
- `assets/` — place to add images and other static assets.

How to run locally

Open `index.html` in your browser directly, or serve with a simple HTTP server (recommended):

```bash
# from the project root
python3 -m http.server 8000
# then open http://localhost:8000
```

Replace the placeholder profile image in `assets/` and update project cards in `index.html` with your actual project links and screenshots.
